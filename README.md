# memviz

Explorateur de memoire vectorielle locale avec UI inspiree de Supabase/Pinecone, interfacable avec Claude.

## Demarrage rapide

```bash
# Backend
cd server
npm install
npm run dev

# Frontend (dans un autre terminal)
cd client
npm install
npm run dev
```

## Tests

```bash
cd server && npm test
cd client && npm test
```
